🗺 ROBO EXPLORER

Your robot is an explorer in an unknown room. The map starts completely dark. As the robot navigates and scans, it reveals the world — just like fog of war in strategy games.

HOW TO PLAY

  • Press F in the drive window to start autonomous line following — the robot follows the black tape on the floor.
  • Every stop marker on the tape triggers a 360° depth scan — this is your CHECKPOINT.
  • Each checkpoint reveals part of the fog map and awards XP.
  • Complete the full path to finish the run.
  • Unlock achievements and level up!

SCORING

  • Checkpoint scan  → +500 XP
  • 360° point cloud → +1500 XP
  • Complete a run  → +800 XP
  • Speed-run bonus  → +1000 XP (under 5 min)

MANUAL MODE (WASD)

  • R key = manual 360° scan → checkpoint XP
  • T key = build point cloud → +1500 XP
